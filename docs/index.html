Über diese Seite

Ich bin kein Designer. Ich bin Techniker und mag es raw, lean, und strebe semantische Perfektion an. Daher sind hier möglichst viel HTML-Elemente selbst und möglichst wenig CSS-Klassennamen zum Einsatz gekommen.

Ich habe bewusst auf ein CSS-Framework (ich bin von der Zurb-Foundation-Fraktion, Bootstrap mag ich nicht) verzichtet und Good Ole HTML benutzt: es ist von Natur aus responsive/adaptive! Zur besseren Lesbarkeit habe ich die Maximalbreite des Inhalts auf 640px gesetzt, Mobilgeräte sind ggf. dann noch etwas schmaler. Auch ist alles in-line definiert (SVGs, die Styles, …): so ist es nur eine einzige Datei die funktioniert as-is.

Die Seite ist statisch über GitHub gehostet und über CloudFlare weiterserviert. Alles fürlau. Meiner Meinung nach „gehört statischen Seiten die Zukunft“.
